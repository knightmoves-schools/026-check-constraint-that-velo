# 026 Check Constraint

[![026 Check Constraint](https://img.youtube.com/vi/wK84wipy4rM/0.jpg)](https://www.youtube.com/watch?v=wK84wipy4rM)

The SQL in the `exercise.sql` file
- Should create a table named `Orders` with the following columns; `ID`, `CUSTOMER_ID`, `TOTAL_AMOUNT`, `ORDER_STATUS`, `SHIPPING_FEE`, `PAYMENT_METHOD`, and `PAID`
- Should make the `CUSTOMER_ID` column unable to repeat values
- Should make the following columns require values; `CUSTOMER_ID`, `TOTAL_AMOUNT`, `ORDER_STATUS`,and `PAYMENT_METHOD`
- Should set the `ORDER_STATUS` value to 'Pending' when left blank
- Should make the `SHIPPING_FEE` unable to have a value lower then 2
- Should make the `TOTAL_AMOUNT` unable to have a negative value
